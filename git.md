## Git

**初始化仓库**

	# cd ~/develop/noobject/java 6 db2
	# git init

**添加文件**
	
	# git add mariadb-java-client-1.2.0.jar
	# git add src
	# git add *	

**本地提交**	

	# git commit -m "init"

**忽略文件**	

	# touch .gitignore
	# vi .gitignore

**查看状态**
	
	#git status
